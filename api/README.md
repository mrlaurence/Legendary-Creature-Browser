## Search API ⁽⁽ଘ( ˊᵕˋ )ଓ⁾⁾

### /search
- Method: GET
- Query format: `/search?n=123&q=shrek%20and%20donkey`
  - `n`: the upper limit on search results
  - `q`: the search query (query escaped)

### /random
- Method: GET
- Query format: `/random`
  - `n`: the number of random creatures to return
